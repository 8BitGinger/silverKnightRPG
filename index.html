Game Description

Silver Knight is a text based RPG game.

You will be playing as a knight who is on a quest to save the Foglands from invading Werewolves all to fulfill your oath as a Silver Knight and save the kingdom.

Video Demo

Installation Instructions

First Clone the Repo from this link:
Silver Knight RPG Repo

Then run the following commands in your terminal:

npm install

then run:

npm run start

Follow the instructions in the terminal to play the game.

Easy to install, easy to play!
